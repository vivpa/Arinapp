# JCL Convention App

A web application for attendees of the Junior Classical League (JCL) National Convention.

## Features

- 🏛️ **Home** — Welcome page with quick convention info
- 📅 **Schedule** — Full 5-day convention schedule with color-coded event types
- ⚔️ **Events & Contests** — Browse all 23 competitions (Academic, Certamen, Creative Arts, Spirit, Athletics) with search and filter
- 📢 **Announcements** — Latest news and updates from convention staff
- 📍 **Venue & Maps** — All venue locations with addresses and travel information
- ℹ️ **About JCL** — Mission, officer list, FAQ, and external links

## Tech Stack

- [React](https://react.dev/) 19 + [Vite](https://vitejs.dev/) 8
- [React Router DOM](https://reactrouter.com/) for client-side navigation
- Custom CSS with a classical Roman theme (Cinzel & Crimson Text fonts)

## Getting Started

```bash
npm install
npm run dev       # start dev server at http://localhost:5173
npm run build     # production build → dist/
npm run preview   # preview production build
npm run lint      # ESLint
```
